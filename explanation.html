Orbital elements for solar system objects

Interactive illustration of the meaning of the orbital elements inclination i, longitude of the ascending node Ω, and the argument of perihelion ω. Use the GUI to change the orbital elements. You can move the GUI around with the mouse.
Keyboard controls:
  • Press 'h' to toggle the visbility of these explanations.
  • Press 'p' to toggle GUI visibility.
  • Press 'a' to toggle the animation of the body moving along the orbit. Note this is done incorrectly by uniformly changing the true anomaly instead of time. The animation is only intended to illustrate prograde vs. retrograde motion.
